We note that if n^4 + 1 is divisible by p^2 with p an odd prime then necessarily n^4 ≡ -1 (mod p^2) and, in particular, n^4 ≡ -1 (mod p). Since the multiplicative group modulo an odd prime p is cyclic of order p – 1, a necessary condition for the congruence n^4 ≡ -1 (mod p) to have a solution is that –1 be a fourth power in the field Z/pZ. This implies that the equation x^4 ≡ -1 (mod p) is soluble only when the number of fourth powers (which is (p – 1)/4) is at least large enough to include –1. In fact, one can show that aside from the trivial case p = 2 (which in this problem does not work because n^4 + 1 is always 1 or 2 mod 4), the prime p must satisfy 8 | (p – 1). Thus 17 is indeed the least prime with the property that there exists a positive integer n (for example n = 2) with n^4 + 1 divisible by p^2.
