We need the probability that a random red–blue coloring of the 8 vertices admits a nontrivial rotation that sends all blue vertices onto originally red ones.  Equivalently, letting B be the set of blue vertices, we require some r≠0 mod 8 with (B+r)∩B=∅.  One shows easily that any B of size 0,1,2,3 works, so that contributes C(8,0)+C(8,1)+C(8,2)+C(8,3)=1+8+28+56=93 colorings.
