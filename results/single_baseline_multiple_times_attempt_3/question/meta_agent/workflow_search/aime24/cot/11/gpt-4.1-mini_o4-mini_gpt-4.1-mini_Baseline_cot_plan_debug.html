We need the number of monotonic paths from (0,0) to (8,8) with exactly four direction changes. Such a path consists of 16 moves: 8 east (E) and 8 north (N). A change in direction occurs at each adjacent pair where the move letter changes. If there are k runs (maximal blocks of identical moves), then there are k−1 changes; we want k−1=4, so k=5 runs.

Each path is a sequence of 5 runs alternating between E and N. There are two cases:

2) Start with N: runs sequence N,E,N,E,N gives 3 N-runs and 2 E-runs, similarly 21×7=147.
